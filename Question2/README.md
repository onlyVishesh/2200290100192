# Stock Price Frontend

Quick React frontend for stock price visualization.

## Setup

1. Install dependencies:

   ```
   npm install
   ```

2. Create `.env` file:

   ```
   REACT_APP_API_URL=http://localhost:3001
   PORT=3000
   ```

3. Start app:

   ```
   npm start
   ```
